## Quick Start

1. clone project

```shell
git clone https://github.com/laohuoyan/randomanimal-generator
```

2. install dependencies

```shell
cd randomanimal-generator
pnpm install
```

3. local development

```shell
pnpm dev
```

open `http://localhost:3000` for preview
